## 📂 Repository Structure

### 1️⃣ 🛠 Technical (Core Development)
- `📁 Backend/` (C#, .NET, SQL Server projects & architecture)
- `📁 Frontend/` (HTML, CSS, JS, React/Angular, etc.)
- `📁 Mobile-Dev/` (MAUI, Flutter, or Native development)
- `📁 Web-General/` (Web fundamentals and full-stack snippets)
- `📁 Problem-Solving/` (Logic, DSA, and LeetCode challenges)
- `📁 Resources/` (Curated links, documentation, and dev tools)
- `📁 College-Resources/` (Computer Science fundamentals and academic tech data)
- `📁 Roadmaps/` (Step-by-step tracks for Software Engineering)

### 2️⃣ 🏗 Community Projects (Open Source)
This is where we build together. Every project here is a collaborative effort by Root Node members.
- `📁 Projects/` (Shared Open-Source applications and tools)

### 3️⃣ 📋 Non-Technical
- `📁 Soft-Skills/` (Leadership, Communication, and Interview prep)
- `📁 Languages/` (English, Spanish, German learning materials)

### 4️⃣ ⏳ Members-Vault (Personal Timers)
- `📁 Members-Vault/Name-Surname/` (Individual progress and daily logs)

---

## 🛠 How to Contribute

1. **Personal Space:** Create your sub-folder inside `📁 Members-Vault/`.
2. **Collaborate:** Check the `📁 Projects/` folder for ongoing community work.
3. **Workflow:** - Maintain clean commits.
   - Open a **Pull Request** to contribute to shared folders.
